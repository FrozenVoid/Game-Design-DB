Install golly https://sourceforge.net/projects/golly/
IN preferences color uncheck "Use gradient" 
it will show the full pallete
Copy any ruleset to clipboard(format is autodetected):
Click File->Open clipboard,
The pattern will load and start running
(Use Control-Start generating if stopped),

If there are errors(pattern can't have lines with indentation), 
the default rules b2/s23 will be used

For random ruleset exploration:
https://frozenvoid.github.io/CellularAutomata/Golly/Rules/Rulegen.html
